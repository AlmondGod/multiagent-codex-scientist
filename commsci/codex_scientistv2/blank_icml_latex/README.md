Vendored ICML 2025 LaTeX support files copied from Sakana AI's
AI-Scientist-v2 `ai_scientist/blank_icml_latex` template directory:

https://github.com/SakanaAI/AI-Scientist-v2/tree/main/ai_scientist/blank_icml_latex

The runner copies these files into each `latex/` output directory before
writing and compiling `paper.tex`. The generated `paper.tex` is project-specific;
the original Sakana template is preserved in run outputs as
`blank_icml_template.tex`.

The Sakana AI-Scientist-v2 repository license is included here as
`LICENSE.Sakana-AI-Scientist-v2`.
